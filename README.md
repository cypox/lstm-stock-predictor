# lstm-stock-predictor

LSTM network to predict stock prices.

Combine indicators and history for better predictions.

# Installation

```bash
python3 -m venv env
pip3 install -r requirements.txt
```

# Running

```bash
source env/bin/activate
python3 src/lsp.py
```
